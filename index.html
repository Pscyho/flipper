Flipper Zero

The Flipper Zero is a portable multi-tool for pentesters and security researchers. It's designed to interact with access control systems, radio protocols, and more. With its toy-like appearance, it's both functional and discreet.

Key Features: RFID/NFC, Sub-1GHz radio, Infrared, GPIO, iButton, BadUSB

Use Cases: Access control testing, RFID cloning, radio signal analysis, infrared remote control

Wi-Fi Deauther

An affordable device based on ESP8266/ESP32 that can perform Wi-Fi penetration testing. It can send deauthentication frames to disconnect devices from networks for testing purposes.

Key Features: Wi-Fi scanning, deauthentication attacks, probe request flooding, MAC spoofing

Use Cases: Network security testing, Wi-Fi penetration testing, educational purposes
